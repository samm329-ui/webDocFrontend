# **App Name**: MediWell Clinic

## Core Features:

- Hero Section: Display doctor information and calls to action using an animated WebP background.
- Appointment Type Selector: Allow users to select the type of appointment (Check-Up or Surgery) which sets appointmentType state and conditionally renders the form.
- Dynamic Appointment Form: Render a form with fields for name, phone, email, preferred date, selected appointment type, and notes; appointment type is read-only; display a static success message on submit.
- About Doctor Section: Show doctor information including image, name, specialization, experience, and qualifications.
- Common Diseases/Treatments: Display common diseases and treatments in a horizontal scrolling list of cards.
- Contact Section: Display clinic address, phone number (tap-to-call), WhatsApp button, and embedded Google Map.
- Footer: Display clinic name, navigation links, and a privacy policy placeholder.

## Style Guidelines:

- Primary color: Soft blue (#74A4BC), evokes trust and calm.
- Background color: Light grey (#F0F4F8), provides a clean, non-distracting backdrop.
- Accent color: Muted green (#A2B86C), complementing blue for a natural, healthy feel.
- Body and headline font: 'PT Sans' sans-serif for a modern, approachable style
- Use simple, line-based icons to represent services and treatments.
- Mobile-first, single-page layout with clear section dividers and consistent spacing.
- Subtle transition animations when scrolling to sections and displaying success messages.
